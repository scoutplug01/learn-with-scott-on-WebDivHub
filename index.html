HTML5 Complete Guide

The backbone of every website

What is HTML?

HTML (HyperText Markup Language) is the standard markup language for creating web pages. It describes the structure and content of a webpage using tags and elements.

01

HTML Document Structure

Every HTML document follows a basic structure with DOCTYPE, html, head, and body tags.

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>My Website</title>
</head>
<body>
    <h1>Hello World!</h1>
</body>
</html>

Explanation:

  • <!DOCTYPE html> - Declares HTML5 document
  • <html> - Root element
  • <head> - Contains metadata
  • <body> - Contains visible content
02

Headings (H1-H6)

HTML provides 6 levels of headings, from h1 (most important) to h6 (least important).

<h1>Main Title</h1>
<h2>Subtitle</h2>
<h3>Section Title</h3>
<h4>Sub-section</h4>
<h5>Minor Heading</h5>
<h6>Smallest Heading</h6>

H1 - Main Title

H2 - Subtitle

H3 - Section

03

Paragraphs & Text

The <p> tag defines a paragraph. HTML provides many text formatting tags.

<p>This is a paragraph.</p>
<strong>Bold text</strong>
<em>Italic text</em>
<mark>Highlighted</mark>
<small>Small text</small>

This is a paragraph.

Bold text | Italic text | Highlighted | Small text
04

Links (Anchor Tags)

The <a> tag creates hyperlinks to other pages or resources.

<!-- External Link -->
<a href="https://google.com" target="_blank">
    Visit Google
</a>

<!-- Internal Link -->
<a href="#contact">Contact Me</a>

<!-- Email Link -->
<a href="mailto:email@example.com">
    Send Email
</a>

Attributes:

  • href - Destination URL
  • target="_blank" - Opens in new tab
  • title - Tooltip text
05

Images

The <img> tag embeds images in HTML documents.

<img src="image.jpg" 
     alt="Description" 
     width="300" 
     height="200">

<!-- With Figure -->
<figure>
    <img src="photo.jpg" alt="Photo">
    <figcaption>Caption</figcaption>
</figure>

Important Attributes:

  • src - Image source path
  • alt - Alternative text (accessibility)
  • width/height - Dimensions
06

Lists

HTML provides ordered (numbered) and unordered (bulleted) lists.

<!-- Unordered List -->
<ul>
    <li>Item 1</li>
    <li>Item 2</li>
</ul>

<!-- Ordered List -->
<ol>
    <li>First</li>
    <li>Second</li>
</ol>
Unordered:
  • Item 1
  • Item 2
Ordered:
  1. First
  2. Second
07

Tables

Tables organize data in rows and columns.

<table>
    <thead>
        <tr>
            <th>Name</th>
            <th>Age</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>John</td>
            <td>25</td>
        </tr>
    </tbody>
</table>

08

Forms & Inputs

Forms collect user input with various input types.

<form action="/submit" method="POST">
    <label for="name">Name:</label>
    <input type="text" id="name" name="name">
    
    <label for="email">Email:</label>
    <input type="email" id="email">
    
    <button type="submit">Submit</button>
</form>

Input Types:

  • text - Text input
  • email - Email validation
  • password - Hidden text
  • checkbox - Multiple selection
  • radio - Single selection
09

Semantic HTML5

Semantic tags give meaning to the content structure.

<header>Site header</header>
<nav>Navigation menu</nav>
<main>
    <article>Blog post</article>
    <aside>Sidebar</aside>
</main>
<footer>Site footer</footer>

Semantic Elements:

  • <header> - Page/section header
  • <nav> - Navigation links
  • <main> - Main content
  • <article> - Independent content
  • <section> - Thematic grouping
  • <aside> - Side content
  • <footer> - Page/section footer
10

Div & Span

Generic containers for grouping and styling content.

<!-- Block-level container -->
<div class="container">
    <p>Content inside div</p>
</div>

<!-- Inline container -->
<p>This is <span class="highlight">highlighted</span> text</p>

Difference:

  • <div> - Block element (takes full width)
  • <span> - Inline element (takes needed width)

CSS3 Complete Guide

Style and design your websites beautifully

What is CSS?

CSS (Cascading Style Sheets) is used to style and layout web pages. It controls colors, fonts, spacing, positioning, and responsive design.

01

CSS Syntax & Selectors

CSS rules consist of selectors and declarations.

/* Element Selector */
h1 {
    color: blue;
    font-size: 24px;
}

/* Class Selector */
.button {
    background: red;
    padding: 10px;
}

/* ID Selector */
#header {
    width: 100%;
}

Selector Types:

  • element - Selects all elements of that type
  • .class - Selects elements with that class
  • #id - Selects element with that ID
02

Colors

CSS provides multiple ways to define colors.

/* Named Colors */
color: red;

/* Hexadecimal */
color: #ff0000;

/* RGB */
color: rgb(255, 0, 0);

/* RGBA (with transparency) */
color: rgba(255, 0, 0, 0.5);

/* HSL */
color: hsl(0, 100%, 50%);
03

Box Model

Every element is a box with content, padding, border, and margin.

.box {
    width: 200px;
    height: 100px;
    padding: 20px;      /* Inside space */
    border: 5px solid black;
    margin: 10px;       /* Outside space */
}
Content
Blue = Content | Space = Padding | Black = Border
04

Flexbox Layout

Flexbox is a one-dimensional layout method for arranging items.

.container {
    display: flex;
    justify-content: space-between;
    align-items: center;
    gap: 20px;
}

.item {
    flex: 1;
}

Key Properties:

  • display: flex - Enable flexbox
  • justify-content - Align horizontally
  • align-items - Align vertically
  • flex-direction - Row or column
  • gap - Space between items
05

CSS Grid

Grid is a two-dimensional layout system.

.grid {
    display: grid;
    grid-template-columns: repeat(3, 1fr);
    gap: 20px;
}

/* Responsive Grid */
.grid {
    grid-template-columns: 
        repeat(auto-fit, minmax(250px, 1fr));
}

2
3
06

Positioning

CSS positioning controls element placement.

/* Static (default) */
position: static;

/* Relative - relative to normal position */
position: relative;
top: 10px;
left: 20px;

/* Absolute - relative to parent */
position: absolute;
top: 0;
right: 0;

/* Fixed - relative to viewport */
position: fixed;
bottom: 20px;

/* Sticky - hybrid */
position: sticky;
top: 0;
07

Animations & Transitions

Create smooth animations with CSS.

/* Transition */
.button {
    transition: all 0.3s ease;
}

.button:hover {
    transform: scale(1.1);
}

/* Keyframe Animation */
@keyframes slide {
    from { left: 0; }
    to { left: 100px; }
}

.box {
    animation: slide 2s infinite;
}
08

Media Queries

Make websites responsive on different screen sizes.

/* Mobile First */
.container {
    width: 100%;
}

/* Tablet */
@media (min-width: 768px) {
    .container {
        width: 750px;
    }
}

/* Desktop */
@media (min-width: 1024px) {
    .container {
        width: 1000px;
    }
}

JavaScript Complete Guide

Make your websites interactive and dynamic

What is JavaScript?

JavaScript is a programming language that adds interactivity to websites. It can update content, control multimedia, animate images, and much more.

01

Variables

Variables store data values.

// var (old way)
var name = "John";

// let (can be changed)
let age = 25;
age = 26;  // OK

// const (cannot be changed)
const PI = 3.14;
// PI = 3; // Error!

Variable Types:

  • var - Function scoped (avoid)
  • let - Block scoped, can change
  • const - Block scoped, cannot change
02

Data Types

JavaScript has different types of data.

// String
let name = "John";

// Number
let age = 25;
let price = 9.99;

// Boolean
let isStudent = true;

// Array
let colors = ["red", "green", "blue"];

// Object
let person = {
    name: "John",
    age: 25
};

// Null & Undefined
let empty = null;
let notDefined;
03

Functions

Functions are reusable blocks of code.

// Function Declaration
function greet(name) {
    return "Hello " + name;
}

// Function Expression
const add = function(a, b) {
    return a + b;
};

// Arrow Function
const multiply = (a, b) => a * b;

// Calling functions
console.log(greet("John"));     // Hello John
console.log(add(5, 3));         // 8
console.log(multiply(4, 2));    // 8
04

DOM Manipulation

Access and modify HTML elements with JavaScript.

// Select elements
const title = document.getElementById('title');
const buttons = document.querySelectorAll('.btn');

// Change content
title.textContent = "New Title";
title.innerHTML = "<strong>Bold</strong>";

// Change styles
title.style.color = "blue";
title.style.fontSize = "24px";

// Add/remove classes
title.classList.add('active');
title.classList.remove('hidden');
title.classList.toggle('dark');

// Create elements
const div = document.createElement('div');
div.textContent = "New div";
document.body.appendChild(div);
05

Event Listeners

Respond to user actions like clicks, typing, etc.

// Click event
button.addEventListener('click', function() {
    alert('Button clicked!');
});

// Input event
input.addEventListener('input', (e) => {
    console.log(e.target.value);
});

// Keypress event
document.addEventListener('keypress', (e) => {
    if (e.key === 'Enter') {
        console.log('Enter pressed');
    }
});

// Mouse events
div.addEventListener('mouseenter', () => {
    div.style.background = 'blue';
});

div.addEventListener('mouseleave', () => {
    div.style.background = 'red';
});
06

Arrays & Array Methods

Arrays store multiple values in a single variable.

const fruits = ['apple', 'banana', 'orange'];

// Access items
console.log(fruits[0]);  // apple

// Add items
fruits.push('grape');        // Add to end
fruits.unshift('mango');     // Add to start

// Remove items
fruits.pop();                // Remove from end
fruits.shift();              // Remove from start

// Array methods
const numbers = [1, 2, 3, 4, 5];

// forEach
numbers.forEach(num => console.log(num));

// map (create new array)
const doubled = numbers.map(num => num * 2);

// filter
const evens = numbers.filter(num => num % 2 === 0);

// find
const found = numbers.find(num => num > 3);
07

If/Else Statements

Make decisions in your code.

const age = 18;

// if statement
if (age >= 18) {
    console.log("Adult");
}

// if-else
if (age >= 18) {
    console.log("Adult");
} else {
    console.log("Minor");
}

// if-else-if
if (age < 13) {
    console.log("Child");
} else if (age < 18) {
    console.log("Teenager");
} else {
    console.log("Adult");
}

// Ternary operator
const status = age >= 18 ? "Adult" : "Minor";

// Switch statement
switch(age) {
    case 18:
        console.log("Just turned adult");
        break;
    case 21:
        console.log("Legal drinking age");
        break;
    default:
        console.log("Regular age");
}
08

Loops

Repeat actions multiple times.

// for loop
for (let i = 0; i < 5; i++) {
    console.log(i);  // 0, 1, 2, 3, 4
}

// while loop
let count = 0;
while (count < 5) {
    console.log(count);
    count++;
}

// do-while loop
let num = 0;
do {
    console.log(num);
    num++;
} while (num < 5);

// for...of (arrays)
const colors = ['red', 'green', 'blue'];
for (const color of colors) {
    console.log(color);
}

// for...in (objects)
const person = {name: 'John', age: 25};
for (const key in person) {
    console.log(key, person[key]);
}
09

Fetch API & Async

Get data from servers/APIs.

// Fetch with promises
fetch('https://api.example.com/data')
    .then(response => response.json())
    .then(data => console.log(data))
    .catch(error => console.error(error));

// Async/Await (modern way)
async function getData() {
    try {
        const response = await fetch('https://api.example.com/data');
        const data = await response.json();
        console.log(data);
    } catch (error) {
        console.error(error);
    }
}

getData();
10

LocalStorage

Save data in the browser.

// Save data
localStorage.setItem('name', 'John');
localStorage.setItem('age', '25');

// Save object (convert to JSON)
const user = {name: 'John', age: 25};
localStorage.setItem('user', JSON.stringify(user));

// Get data
const name = localStorage.getItem('name');
console.log(name);  // John

// Get object (parse JSON)
const savedUser = JSON.parse(localStorage.getItem('user'));
console.log(savedUser.name);  // John

// Remove data
localStorage.removeItem('name');

// Clear all
localStorage.clear();

// Get data
const name = localStorage.getItem('name');
console.log(name);  // John

// Get object (parse JSON)
const savedUser = JSON.parse(localStorage.getItem('user'));
console.log(savedUser.name);  // John

// Remove data
localStorage.removeItem('name');

// Clear all
localStorage.clear();
